## 🎯 Présentation

**MikaBio** est une application ERP de gestion d'entreprise conçue avec les dernières versions du stack React/Next.js :

- **Next.js 16** (App Router, Server Components)
- **React 19** (dernière version stable)
- **Tailwind CSS 4.0** (bleeding-edge)
- **TypeScript 5.7**

## ✨ Modules

- 📦 Gestion des stocks et inventaires
- 👥 Gestion des utilisateurs et rôles
- 📊 Tableaux de bord et rapports
- 🔐 Authentification sécurisée

## 🚀 Lancement rapide

```bash
git clone https://github.com/elinho90/mikabio-frontend.git
cd mikabio-frontend
npm install
npm run dev
```

L'application sera disponible sur `http://localhost:3000`
